# ✈️ Бот мониторинга авиабилетов LED

**@Airdrey_bot** — бот для отслеживания цен на авиабилеты из Санкт-Петербурга.

## Для чего сделан

Бот автоматически следит за минимальными ценами на авиабилеты из Пулково (LED) в 23 направления и присылает отчёты, когда появляются выгодные предложения или цены падают.

## Возможности

### 📊 Автоотчёты (2 раза в день)
Каждое утро в 07:00 и днём в 13:00 МСК бот присылает сводку:
- Актуальные минимальные цены по всем направлениям
- Сравнение со средней ценой за неделю (▲ ▼ →)
- Аномалии: 🔥 если цена упала ≥20%, 🏆 если достигнут исторический минимум

### 🤖 Команды (доступны 24/7)

| Команда | Описание | Пример |
|---------|----------|--------|
| `/today` | Текущие цены по всем направлениям | `/today` |
| `/history <направление> [дней]` | График цен за период | `/history Москва 7` |
| `/alerts` | Текущие аномалии и скидки | `/alerts` |
| `/help` | Справка по командам | `/help` |
| `/start` | Приветствие и chat_id | `/start` |

## Как пользоваться

1. Найди бота в Telegram: **@Airdrey_bot**
2. Отправь `/start` — бот пришлёт приветствие
3. Используй команды для проверки цен в любой момент
4. Автоотчёты приходят сами по расписанию

## Откуда цены

Цены получаются через API Aviasales (Travelpayouts) — это реальные данные по продаже билетов, обновляемые в реальном времени.

## Для кого

- Для тех, кто часто летает из Питера и хочет ловить дешёвые билеты
- Для планирования отпуска — следить за ценами на нужное направление
- Для тех, кто хочет получать алерты, когда появляются горящие предложения
